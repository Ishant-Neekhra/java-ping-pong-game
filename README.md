# 🏓 Java Ping Pong Game

A simple Ping Pong game built using Java Swing.

This project was created as part of my Java workshop learning journey and later customized with additional improvements.

## 🚀 Features

- Paddle movement using keyboard controls
- Ball collision detection
- Score tracking
- High score tracking
- Start, Stop and Restart buttons
- Game Over screen

## 🛠️ Technologies Used

- Java
- Java Swing
- AWT

## 🎮 Controls

| Key | Action |
|------|---------|
| ← | Move Left |
| → | Move Right |

## 📚 What I Learned

- Java GUI development
- Swing components
- Event handling
- Keyboard listeners
- Timers and game loops
- Collision detection
- Object-oriented programming concepts
